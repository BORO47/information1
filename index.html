СПИСОК ВАЖНЫХ ПЛАГИНОВ НА СЕРВЕР

  • ChestCommands - с помощью этого плагина вы можете создать меню
  • PermissionsEx - с помощью этого плагина вы можете донат, и права для этого доната
  • Essentials - с помощью этого плагина вы можете улучшить плагин PermissionsEx
  • EssentialsSpawn - с помощью этого плагина вы можете добавить точку появления
  • ChatEX - с помощью этого плагина вы можете защитить свой сервер от спамеров, пиарщиков, и добавить префиксы
  • NameTagEdit - с помощью этого плагина вы можете добавить префикс в табе и над головой
  • ViaVersion - с помощью этого плагина вы можете сделать так чтобы можно было заходить на сервер с новых версий, например 1.18.1
  • AuthMe - с помощью этого плагина вы можете добавить регистрацию на ваш сервер(не стоит писать на всех серверах один и тот же пароль)
  • SkinRestorer - с помощью этого плагина вы можете добавить красивые скины по нику(например= skin mag1ster)
  • WorldEdit - с помощью этого плагина вы можете изменять мир деревянным топориком
  • WorldGuard - с помощью этого плагина вы можете добавить приваты на свой сервер
